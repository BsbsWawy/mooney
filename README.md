# mooney
Simple CI system written in C programming language

Prerequisites:

1. Install git: https://git-scm.com/downloads
2. Install chocolately: https://chocolatey.org/install
3. Install gcc:
```commandLine
choco install mingw
```
4. Install make:
```commandLine
choco install make
```
5. Install VSCode: https://code.visualstudio.com
6. Run make:
```commandLine
make all
```
